# Портфолио лабораторных работ - Сагор (Sagor)

Этот репозиторий содержит отчеты по курсу "Программирование на Python 2026". Сайт построен с использованием генератора статических сайтов MkDocs.

## Обоснование выбора темы
Для оформления сайта я выбрал тему Material for MkDocs. 
Причины выбора:
1. Современный дизайн: Тема выглядит профессионально и чисто.
2. Адаптивность: Сайт отлично отображается как на ПК, так и на мобильных устройствах.
3. Функциональность: Удобная навигация, поддержка темного режима и отличный поиск.
