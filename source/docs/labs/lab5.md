#### 3.1. Предварительная обработка и обучение базовых моделей
Данные были разделены на обучающую и тестовую выборки. В качестве целевой переменной выступала цена недвижимости (`Целевая.Цена`).

```python
# Краткий фрагмент кода обучения моделей
from sklearn import linear_model, ensemble

# Линейная регрессия
lr_model = linear_model.LinearRegression()
lr_model.fit(X_train, y_train)

# Случайный лес
rf_model = ensemble.RandomForestRegressor(random_state=42)
rf_model.fit(X_train, y_train)
```

#### 3.4. Интеграция с веб-сервисом (Алгоритм)
Для развертывания модели в качестве веб-сервиса (на базе FastAPI) используется следующий алгоритм:
1. **Сериализация:** Сохранение модели в файл `.pkl` через `joblib`.
2. **Создание API:** Инициализация FastAPI приложения и создание POST-эндпоинта `/predict`.
3. **Валидация:** Описание схемы входных данных с помощью `Pydantic`.
4. **Предсказание:** Загрузка модели в память сервера и передача входных данных в `model.predict()`.
5. **Деплой:** Запуск сервиса через сервер `uvicorn`.
