# Отчет по лабораторной работе №2: Основы NumPy и векторные операции

## 1. Цель работы
Изучить основы работы с библиотекой NumPy для языка Python, освоить приемы векторизации для выполнения математических операций без использования циклов, а также научиться проводить статистический анализ и визуализацию данных.

## 2. Задание
- Создать и обработать массивы NumPy.
- Реализовать векторные и матричные операции (сложение, умножение, скалярное произведение, решение СЛАУ, нахождение обратной матрицы и определителя).
- Загрузить данные из CSV-файла и провести их статистический анализ.
- Выполнить Min-Max нормализацию данных.
- Визуализировать результаты с помощью Matplotlib и Seaborn (гистограмма, тепловая карта, линейный график).
- Организовать структуру проекта с разделением кода и тестов (PEP-8, PEP-484, PEP-257).

## 3. Ход работы / Код
Работа была выполнена с использованием среды разработки VS Code. Весь код разбит на два файла:
- main.py — основные функции для вычислений и визуализации.
- test.py — тесты для проверки корректности реализованных функций.

Проект был загружен на GitHub: [https://github.com/sagorgit/numpy_lab](https://github.com/sagorgit/numpy_lab)

### Ключевые этапы:
1. Векторизация: Использование встроенных методов NumPy позволило избежать громоздких циклов for.
2. Типизация: Добавлены аннотации типов (PEP-484) для повышения читаемости кода.
3. Тестирование: Все функции прошли проверку с использованием pytest (17 тестов успешно выполнены).
4. Визуализация: Построенные графики автоматически сохраняются в директорию plots/.

## 4. Выводы
В ходе выполнения лабораторной работы я закрепил навыки работы с NumPy. Стало очевидно, что использование векторизованных операций значительно ускоряет вычисления и делает код более компактным по сравнению с обычными циклами Python. Также я освоил принципы оформления кода согласно стандартам PEP, что делает программное обеспечение профессиональным и удобным для поддержки.
